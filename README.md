# Scabble Share

Timed game similar to scrabble.

## Play game here!

[Scrabble Share](https://scrabble-share.firebaseapp.com/).

### Getting Started

Create a new game and share the givin link with a friend.

### Instaling

```
yarn install
```

or

```
npm i
```

### To Run

```
yarn start
```

or

```
npm run start
```
